🗺️

오픈월드 탐험

설원, 사막, 정글, 동굴과 던전까지 — 광활한 대륙을 자유롭게 누비세요. 하늘을 나는 비행선도 있습니다.

⚔️

실시간 전투

검, 도끼, 활, 지팡이 등 무기마다 고유한 스킬과 콤보. 거대 보스와의 전투에 도전하세요.

⚒️

제작과 교역

재료를 모아 장비를 만들고, 마을 NPC와 거래하며 더 강한 장비로 성장하세요.

🤝

협동 멀티플레이

친구들과 같은 서버에서 함께 사냥하고, 채팅하고, 모험하세요. 모든 진행은 서버에 저장됩니다.

3분 만에 시작하기

런처가 설치부터 업데이트까지 전부 알아서 해줍니다.

1

런처 다운로드

VoxWorldLauncher.exe를 받아 실행하세요. 설치 과정 없이 바로 켜집니다.

2

설치하기 클릭

런처가 최신 버전의 게임을 자동으로 다운로드합니다. 이후 업데이트가 나와도 런처가 알아서 갱신해요.

3

게임 시작

서버 주소가 자동으로 입력된 상태로 게임이 켜집니다. 닉네임을 정하고 캐릭터를 만들어 바로 접속하세요.

💡 Windows SmartScreen 경고가 뜨나요? 코드 서명이 없는 오픈소스 빌드라 표시되는 안내입니다. 추가 정보 → 실행을 누르면 정상적으로 실행됩니다.

공식 서버

런처를 쓰면 자동으로 접속됩니다. 클라이언트에 직접 입력할 때만 이 주소를 사용하세요.

viaduct.proxy.rlwy.net:36453
